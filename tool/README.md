# tool

#### 项目介绍
给pdf增加表单域，支持中文，如需之前其他字体只需将ttf文件放入到项目的font文件夹下即可。

#### 软件架构

1、引用了itext7-core的最新jar，fastjson，commons-lang3。
2、将spring中的ResourceUtils拷贝出来，方便快速获取项目的classpath路径。
